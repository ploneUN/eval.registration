eval.registration Installation
------------------------------

To install eval.registration using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``eval.registration`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        eval.registration

* Re-run buildout, e.g. with:

    $ ./bin/buildout
